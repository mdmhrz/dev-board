‚ùì What is the difference between innerHTML, innerText, and textContent?

  • innerHTML: Gets or sets the HTML content inside an element, including any HTML tags.
  • innerText: Gets or sets the visible text content of an element, excluding hidden elements and respecting CSS styling.
  • textContent: Gets or sets the text content of an element, including hidden elements, but excluding HTML tags.

‚ùì What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element to manage events for all of its child elements. This helps improve performance by reducing the number of event listeners.

‚ùì What is event bubbling in the DOM?

Event bubbling is a mechanism where an event triggered on a child element propagates up through its ancestors in the DOM tree, allowing parent elements to handle events that occur on their child elements.

‚ùì How do you create, add, and remove elements using JavaScript?

  • Create: Use document.createElement('tagName') to create a new element.
  • Append: Use appendChild() to add it to the DOM.
  • Remove: Use removeChild() or element.remove() to delete an element.
